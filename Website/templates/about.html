The Project:

The project uses two supervised machine learning algorithms: RandomForestClassifier and RandomForestRegressor.

The goal is to develop a website using Flask to predict winning teams for the European championship and fuel consumption per 100km in the city.

About the Dataset

We have two datasets:

  • Fuel consumption dataset
  • European championship dataset in soccer

We are aiming to predict the winning teams for the European championship and fuel consumption per 100km in the city using these datasets.

How to Install This Project?

You will need to have VSCode installed on your machine.

This repository provides a requirements.txt file.

Once you have VSCode installed, you can either download the ZIP copy of the project or clone the project using Git.

To install all the required packages for the project to run, run the following command in your terminal:

pip install -r requirements.txt
